Notes for adding more projects:
===============================

For Egyptian projects, use these image options:
- /public/images/egyptian-relief-1.jpg (stone relief)
- /public/images/egyptian-painting-2.jpg (tomb painting)  
- /public/images/egyptian-papyrus-3.jpg (papyrus manuscript)

For other script types, you'll need to add appropriate images:
- Cuneiform projects: cuneiform-tablet.jpg
- Chinese projects: chinese-bamboo.jpg or chinese-silk.jpg
- Anatolian projects: anatolian-hieroglyphs.jpg

Format for new entries:
-----------------------
Project ID: [unique_identifier]
Name: [Display Name]
Author(s): [Author Name(s)]
Image: /public/images/[filename.jpg]
Description: [Brief description of the project]
